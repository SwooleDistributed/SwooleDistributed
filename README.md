可以通过composer安装

autoload必须要指定app和test。
```
{
  "require": {
    "tmtbe/swooledistributed":">2.0.0"
  },
 "autoload": {
    "psr-4": {
      "app\\": "src/app",
      "test\\": "src/test"
    }
  }
}
```
然后在根目录（vendor上级目录）执行下面代码
```
php vendor/tmtbe/swooledistributed/src/Install.php
```
安装结束可以在bin中执行服务器。
